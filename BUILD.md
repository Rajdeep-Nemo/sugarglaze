# Build Instructions

This document explains how to build the project locally.

## Prerequisites

* Go >= 1.21.6
* Git

## Clone the Repository

```bash
git clone https://github.com/Rajdeep-Nemo/pluesi
cd pluesi
```

## Build

```bash
# Build the project
go build ./...
```

## Run

```bash
# Run the compiled binary
./pluesi
```

## Tests

```bash
# Run all tests
go test ./...
```
