# Security Policy

We take security seriously. If you discover a vulnerability, please follow the process below.

---

## Reporting a Vulnerability

**Do not open a public issue.** Instead, report it privately by opening a [GitHub Security Advisory](https://github.com/Rajdeep-Nemo/sugarglaze/security/advisories/new).

Alternatively, you can reach us directly via email:
- rajdeepnemo@gmail.com
- sujaypaul892@gmail.com

Your report should include:
- A clear description of the vulnerability.
- Steps to reproduce the issue.
- The potential impact or attack scenario.

---

## Response

- All reports will be reviewed and acknowledged promptly.
- A fix will be issued as soon as possible depending on severity.
- Responsible disclosure is appreciated and will be credited where appropriate.

---

## Scope

This policy applies to all code within this repository.
